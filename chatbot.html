Chat bot

People use messaging apps more than social networks these days. Messaging is also the preferred method of contacting a business, compared to e-mail & phone.

Give users new ways to interact with their product by building engaging text-based conversational apps.

Objective

To create a chat bot for hospitality related queries in hotels during stay.

Architecture

Technology used: -

  1. Microsoft Bot Framework
  2. Microsoft Application registration portal
  3. Microsoft Bot Builder SDK for Node JS
  4. Google API.AI as NLP and ML
  5. Webhook
  6. Heroku (can be use any other cloud environment like Azure Appservice)
  7. Google Firebase (can be use any other webhook like Azure Webhook)

 

 UseCases

  1. Cab Booking - Ex: I need cab, i want to book a cab
  2. Sight Seeing - Ex: suggest me favorite place to roam around
  3. Request Utility - Ex: I need Soap, Please help me with Extra Pillow
  4. Order Food - Ex: I am hungry

How to use

To enable chatbox, you can click on the pink chat icon on right bottom corner.
